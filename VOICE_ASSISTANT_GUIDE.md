# CROPXAI Voice Assistant Guide

## Overview
The CROPXAI Voice Assistant allows farmers to interact with the system hands-free using voice commands in English, Telugu, and Hindi.

## How to Use

### 1. Activate Voice Assistant
- Click the 🎤 microphone button in the dashboard header
- Or click the voice button in the welcome section

### 2. Start Listening
- Click "Start Listening" button in the voice modal
- Speak clearly into your device's microphone
- The system will transcribe your speech in real-time

### 3. Supported Voice Commands

## Technical Requirements

### Browser Support
- **Best Support:** Google Chrome, Microsoft Edge
- **Good Support:** Firefox, Safari (with limitations)
- **Required:** Microphone access permission

### Language Settings
The voice assistant automatically uses the language selected in the language dropdown:
- English (en-US)
- Telugu (te-IN)
- Hindi (hi-IN)

## Tips for Best Results

### 1. Microphone Setup
- Use a good quality microphone
- Minimize background noise
- Speak clearly and at a moderate pace
- Position microphone 6-12 inches from your mouth

### 2. Command Clarity
- Use simple, direct commands
- Speak one command at a time
- Wait for processing to complete before next command
- Use keywords from the supported commands list

### 3. Environment
- Quiet environment works best
- Avoid echo or reverb
- Close windows to reduce outside noise
- Turn off fans or AC if possible

## Troubleshooting

### Voice Not Recognized
**Problem:** System doesn't respond to voice
**Solutions:**
- Check microphone permissions in browser
- Ensure microphone is not muted
- Try refreshing the page
- Check if browser supports speech recognition

### Wrong Language Detected
**Problem:** Commands in one language interpreted as another
**Solutions:**
- Select correct language from dropdown first
- Speak more clearly
- Use language-specific keywords
- Restart voice assistant

### Commands Not Working
**Problem:** Commands recognized but no action
**Solutions:**
- Ensure you're logged in
- Fill required form fields first (for auto-detect)
- Use exact command phrases
- Check help command for available options

## Privacy & Security

### Data Handling
- Voice data is processed locally in your browser
- No voice recordings are stored on servers
- Speech recognition uses browser's built-in API
- All data remains on your device

### Permissions
- Microphone access required
- Permission requested only when voice assistant is activated
- Can be revoked anytime from browser settings
